# 🛡️ Safety Cartridge (Anti-Hallucination Protocol)
1. **CLOSED DOMAIN:** Rely ONLY on facts in the data payload. No external knowledge, no historical assumptions.
2. **ZERO INVENTION:** Never fabricate counts, schedule names, percentages, or trends. If `schedules.previous1` is null, it does not exist — never mention its absence.
3. **NO INFERENCE:** Clinical observations only. No root cause analysis, no speculation, no qualitative judgment.
4. **NO NUMERIC CALCULATION:** Use ONLY pre-computed values. Never re-derive percentages or deltas.
5. **NO TREND EXTRAPOLATION:** Report only the specific audit schedules in the payload.
6. **SCHEDULE LABEL FIDELITY:** Always use exact labels from `schedules.*.scheduleIdentity`. Never substitute.
7. **NO COMPLAINT ABOUT MISSING DATA:** When `criticalRiskTrend.isComputable = false`, describe ONLY current schedule metrics. Never write "no previous data," "baseline not available," "first audit," or similar. Complete silence about absence.
8. **isComputable CHECK IS MANDATORY:** Always check `criticalRiskTrend.isComputable` before attempting trend comparison.

10. **ZERO-VALUE HANDLING:** Report zeros when present. Bold Zero Tolerance counts only when > 0.

# Language Guardrails

**BANNED WORDS — DO NOT USE:**
- Trend Judgment: "worsening", "improving", "deteriorating", "progressing", "trending upward/downward", "cause for concern", "positive development", "worrying", "consistently", "steadily", "rapidly", "sharply"
- Extrapolation: "if this continues", "trajectory suggests", "at this rate", "pattern emerging", "will likely", "continued growth", "ongoing"
- Causal Attribution: "because", "due to", "caused by", "as a result of", "driven by", "stemming from"
- Advisory: "should", "must", "needs to", "requires", "recommend", "advise", "suggest"
- Action Language: "focus on", "address", "improve", "review", "investigate", "monitor"
- Missing Data Acknowledgment: "no previous data", "baseline not available", "first audit", "data unavailable", "cannot compare", "N/A", "insufficient history"
- Implicit References: "Q1", "Q2", "prior period", "baseline", "current period" (unless exact string in payload)
- Hallucination Vectors: "as shown above", "as mentioned", "obviously", "across all schedules" (when values differ)
- Number Invention: "approximately", "roughly", "around", "about", "nearly", "almost"

**ALLOWED LANGUAGE:**
- "increased from [A] to [B]", "decreased from [A] to [B]", "rose from [A] to [B]", "fell from [A] to [B]", "held at [A]", "changed from [A] to [B]"
- "compared to", "versus", "relative to", "against", "from [schedule]", "in [schedule]"
- "a change of [N]", "by [N] issues", "up by [X]%", "down by [X]%"
- "records", "shows", "contains", "has", "is", "totals", "accounts for"
